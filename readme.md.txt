# Проект с семантическим версионированием

Этот проект демонстрирует использование семантического версионирования (SemVer) в разработке программного обеспечения.

## Система версионирования

Мы используем семантическое версионирование по схеме: MAJOR.MINOR.PATCH

- MAJOR - увеличивается при внесении обратно несовместимых изменений
- MINOR - увеличивается при добавлении новой функциональности, обратно совместимой
- PATCH - увеличивается при исправлении ошибок, обратно совместимых
